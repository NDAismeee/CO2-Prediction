# CO2-Prediction

## Ghi chú về dữ liệu

Phần dữ liệu sử dụng trong source code là **dữ liệu niên giám theo năm do phía Thanh Hóa cung cấp**, bao gồm thông tin đến **năm 2024**.  
Bộ dữ liệu này được dùng làm **nguồn đầu vào để huấn luyện và dự đoán lượng phát thải CO₂ cho các năm tiếp theo (2025–2029)**.

Ngoài ra, dự án còn có **bộ dữ liệu dự đoán sử dụng ảnh vệ tinh**, được cập nhật đến **năm 2025**.  
Tuy nhiên, **các file ảnh vệ tinh có dung lượng rất lớn** nên **không được upload trực tiếp lên hệ thống quản lý mã nguồn GitHub**.  
Nếu cần sử dụng hoặc kiểm tra bộ dữ liệu này, vui lòng liên hệ trực tiếp với nhóm phát triển để được cung cấp.
